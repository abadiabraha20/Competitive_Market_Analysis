# Bayesian Calibration of Replicator Dynamics for Competitive Market Analysis

This repository contains all Python code for reproducing simulations, figures, and numerical results in the manuscript:

**"Bayesian Calibration of Replicator Dynamics for Competitive Market Analysis"** (Applied Mathematics and Computation)

## Contents

- Adaptive Metropolis-Hastings MCMC for parameter estimation (20 parameters, 4 chains, 500k iterations)
- Bifurcation analysis (transcritical and Hopf bifurcations) with uncertainty propagation
- Global sensitivity analysis (Sobol indices, Morris screening)
- Strategic risk quantification (Value-at-Risk, conditional VaR, exclusion probability contours)
- All 10 figures from the manuscript

## Requirements

- Python 3.8+
- NumPy, SciPy, Matplotlib, Pandas, ArviZ
